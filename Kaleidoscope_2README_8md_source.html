1 # Kaleidoscope
2 
3 Flexible firmware for Arduino-powered keyboards.
4 
5 This package contains the "core" of Kaleidoscope and an [example firmware "Sketch."](https://github.com/keyboardio/Kaleidoscope/blob/master/examples/Kaleidoscope/Kaleidoscope.ino)
6 
7 If you're just getting started with the Keyboardio Model 01, the [introductory docs are here](https://github.com/keyboardio/Kaleidoscope/wiki/Keyboardio-Model-01-Introduction) and the source for the basic firmware package is here: https://github.com/keyboardio/Model01-Firmware. It's probably a good idea to start there, learn how to modify your keymap and maybe turn some modules on or off, and then come back to the full repository when you have more complex changes in mind.

9 # Getting Started
10 
11 Pick a directory to work in, you'll need to clone multiple repositories. We'll assume you picked `$HOME/kaleidoscope`, if you chose another adapt the commands below accordingly.
12 
13 ## Setup the Arduino IDE
14 
15 Setup the Arduino IDE on your system. Make sure you install at least version 1.6, since older version may not support all required features.
16 
17 * On Linux, follow the instructions [on the wiki](https://github.com/keyboardio/Kaleidoscope/wiki/Arduino-Setup-Linux).
18 * On macOS, install using [homebrew](http://brew.sh/) [cask](https://caskroom.github.io/) with `brew cask install arduino` or download the application from [the official website](https://www.arduino.cc/en/Main/Software) and move it to your `/Applications` folder.
19 
20 ## Install the Kaleidoscope Hardware Definitions
21 
22 ```sh
23 # you'll need to know your arduino sketchbook directory
24 
25 # on macOS the default is
26 SKETCHBOOK_DIR=$HOME/Documents/Arduino
27 # on Linux the default is
28 SKETCHBOOK_DIR=$HOME/Arduino
29 
30 # go to your kaleidoscope directory
31 cd $HOME/kaleidoscope
32 # then clone the hardware definitions
33 git clone --recursive https://github.com/keyboardio/Arduino-Boards.git
34 # and make them available to the arduino environment
35 mkdir -p $SKETCHBOOK_DIR/hardware/keyboardio
36 ln -s $HOME/kaleidoscope/Arduino-Boards $SKETCHBOOK_DIR/hardware/keyboardio/avr
37 ```
38 
39 ## Clone and Build the Kaleidoscope Firmware
40 
41 ```sh
42 # go to your keyboardio directory
43 cd $HOME/kaleidoscope
44 # then clone the firmware repository
45 git clone https://github.com/keyboardio/Kaleidoscope.git
46 # and build your firmware!
47 cd Kaleidoscope
48 make
49 ```
